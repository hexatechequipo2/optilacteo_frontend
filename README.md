# OptiLácteo — Frontend

OptiLácteo es una plataforma SaaS multi-tenant para la gestión y monitoreo de procesos productivos en PyMEs de la industria láctea. Integra control de calidad, trazabilidad de lotes, monitoreo IoT en tiempo real y recomendaciones basadas en Machine Learning.

Este repositorio contiene el frontend del sistema, desarrollado en React con TypeScript y Tailwind CSS.
